You will find here all the material related to the book [**Introduction to Bayesian statistics with R**](https://oliviergimenez.github.io/introduction-to-bayesian-statistics-with-R) written by Olivier Gimenez, and more specifically:

-   The `R Markdown` files that contain the `R` code and the text to reproduce the different chapters: [`index.Rmd`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/index.Rmd), [`01-principles.Rmd`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/01-principles.Rmd), [`02-mcmcmethods.Rmd`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/02-mcmcmethods.Rmd), [`03-implementation.Rmd`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/03-implementation.Rmd), [`04-priors.Rmd`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/04-priors.Rmd), [`05-regression.Rmd`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/05-regression.Rmd), [`06-glms.Rmd`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/06-glms.Rmd) and [`07-conclusions.Rmd`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/07-conclusions.Rmd).

-   The `R` scripts cleaned of the text to allow you to run the code more easily: [`scriptsR.zip`](https://github.com/oliviergimenez/introduction-to-bayesian-statistics-with-R/raw/refs/heads/main/scriptsR.zip).
